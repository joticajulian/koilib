Class Serializer

The serializer class serialize and deserialize data using protocol buffers.

NOTE: This class uses the protobufjs/light library internally, which uses reflection (use of eval and new Function) for the construction of the types. This could cause issues in environments where eval is not allowed, like in browser extensions. In such cases, this class must be confined in a sandbox environment where eval is allowed. This is the principal reason of having the serializer in a separate class.

example
const descriptorJson = {
  nested: {
    awesomepackage: {
      nested: {
        AwesomeMessage: {
          fields: {
            awesomeField: {
              type: "string",
              id: 1
            }
          }
        }
      }
    }
  }
}
const serializer = new Serializer(descriptorJson)
